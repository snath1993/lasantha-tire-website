# Media Publisher Job - Example Caption Files

This folder contains example caption files for the Media Publisher system.

## How Captions Work

1. **Specific Caption**: `<filename>-caption.txt`
   - Example: `my-video.mp4` uses `my-video-caption.txt`

2. **Generic Caption**: `caption.txt`
   - Used for all files without specific caption

3. **Auto-Generated**: If no caption files exist
   - System generates default Sinhala caption
